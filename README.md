# Earthsong Theme for Zed

Warm, earthy Zed theme based on Dayle Rees' original Earthsong color scheme.

## Repo Layout

- `extension.toml`: Zed extension manifest.
- `themes/earthsong.json`: theme family JSON consumed by Zed.
- `LICENSE`: upstream MIT license from Dayle Rees' `colour-schemes` repository.

## Test Locally In Zed

1. Open Zed.
2. Run `zed: extensions`.
3. Click `Install Dev Extension`.
4. Pick this repository folder.
5. Open the theme selector and choose `Earthsong`.
